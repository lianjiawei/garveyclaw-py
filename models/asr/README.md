# Local ASR Models

WeClaw ships with a small default Chinese Vosk model so new users can try local voice input without a separate model download.

Default Vosk model path:

```text
models/asr/vosk-model-small-cn-0.22
```

Source: https://alphacephei.com/vosk/models

Bundled model: `vosk-model-small-cn-0.22`

License: Apache 2.0, as listed on the official Vosk model page.

Other downloaded models are intentionally ignored by Git because they can be large and machine-specific.
